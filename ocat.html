NAME

ocat - dump Oberon text file to standard out

SYNOPSIS

ocat documentname | filepath

DESCRIPTION

ocat reads the specified Oberon text document, which is either stored in the Oberon V4 binary format or as ASCII text, and displays its ASCII text on the standard output device.

ENVIRONMENT

OBERON

a colon-separated list of directories in which ocat searches for the text document unless an explicit file path is provided. If OBERON is not defined, files are only searched in the current working directory. The environment variable OBERON may be set e.g.Â under the bash shell by

% export OBERON=.:Mylib:OberonLib
